## ­Ъџђ рІІріЊ рІІріЊ рЅБрѕЁрѕфрІФрЅх (Features)
* **рІерЅ░рѕЏрѕфрІјрЅй рѕЮрІЮрїѕрЅБ:** рІерЅ░рѕЏрѕфрІјрЅйріЋ рѕЎрѕЅ рѕхрѕЮрЇБ рІ▓рЇЊрѕГрЅхрѕўріЋрЅх ріЦріЊ рѕхрѕЇріГ рЅЂрїЦрѕГ рѕўрѕўрІЮрїѕрЅЦрЇб
* **рІеріГрЇЇрІФ ріГрЅхрЅхрѕЇ:** рЅарЅ┤рѕї рЅаріЕрѕЇ рІерѕџрІ░рѕерїЅ ріГрЇЇрІФрІјрЅйріЋ ріерЅ░рѕЏрѕфрІЇ рѕўрЅ│рІѕрЅѓрІФ рїІрѕГ рЅарѕЏрїѕріЊріўрЅх рѕўрѕўрІЮрїѕрЅЦрЇб
* **CRUD Operations:** рІ│рЅ│ріЋ рѕўрЇЇрїарѕГрЇБ рѕЏріЋрЅарЅЦрЇБ рѕЏрІ░рѕх ріЦріЊ рѕўрѕ░рѕерІЮ рІФрѕхрЅйрѕІрѕЇрЇб

## ­ЪЏа рІерЅ░рїарЅђрѕЎ рЅ┤ріГріќрѕјрїѓрІјрЅй (Tech Stack)
* **Frontend:** React.js
* **Backend:** Node.js & Express
* **Database:** PostgreSQL (Hosted on Supabase)
* **Hosting:** Vercel (Frontend) & Render (Backend)
